数据转换类

1、字符串转化为时间

str_to_date(str,formate)

SELECT STR_TO_DATE(arrv_dt,"%Y-%m-%d %H:%I:%s") from xml_flight_info

操作类

1、将筛选信息导入新建的表

不同于sqlserver这里没有直接into的操作，应用create语法创建新表从而将筛选的数据导入新的表中。
CREATE TABLE xml_flight_info_backUp (select * from xml_flight_info WHERE LENGTH(arrv_dt)=19)
