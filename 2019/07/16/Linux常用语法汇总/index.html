Linux常用语法汇总

2019-07-16

文件处理语法

cd——切换到指定的目录下，跟windos是一样的

程序部署语法

java -jar xx.jar   ————运行jar包<br>
sudo nohup & 程序   ————xshell退出后继续运行
sudo fuser -k -n tcp 80 ——————强制杀死某个端口的进程
nohup java -jar xxx.jar >soringboot.log 2>&1&
————没有写入权限的文件夹重新定向
netstat –apn
————查看所有的进程及端口号
