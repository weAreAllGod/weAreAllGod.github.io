anoconda创建虚拟环境

2019-08-02

anoconda是个非常好用的工具，不对，应该说是一个好用的环境集成工具，但是在使用的时候我们总会由于版本问题遇见这样活着那样的问题。比如说，python3.7对某些用3.6写的程序就会报一些莫名的错误，所以就希望把python的版本调到3.6，其实我们是可以将anoconda中的内容进行覆盖，但是如果每次都这样覆盖过来覆盖过去，就需要重新安装一些包，比较容易出错，所以我们可以在anoconda中创建虚拟环境来解决这个问题。方法如下

1、打开anaconda prompt运行命令conda create -n my3.7，my3.7是我的环境名，可以随意命名。
2、在弹出来的询问框里输入y

在我的anconda安装文件夹下的envs下多了一个my3.7的路径，这就是我们刚才创建的新的环境，如下图：

在prompt中激活这个环境，并在这个环境中安装python3.7,命令如下

1、conda activate my3.7
2、conda install python=3.7

最后在编辑器里将编译器设为我们改文件夹下的python就行了
